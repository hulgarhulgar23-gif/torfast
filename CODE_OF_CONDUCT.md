# Code of Conduct

This project wants clear, respectful, reviewable collaboration.

## Expected behavior

- Be respectful.
- Critique ideas and changes, not people.
- Keep claims tied to code, tests, docs, or saved proof.
- Accept correction quickly when privacy, safety, or compatibility risks are found.

## Unacceptable behavior

- Harassment, insults, or personal attacks.
- Dismissing privacy or safety concerns without evidence.
- Sharing private or sensitive information in public issues.

## Reporting

If someone breaks these rules, contact the maintainer privately instead of
starting a public fight.
